# README



## 開発言語

* Ruby 2.6.5
* Rails 5.2.5



## 就業Termの技術

* devise
* フォロー機能



## カリキュラム外の技術

* Google API



## 実行手順

```shell
$ git clone git@github.com:sachio-portfolio/prodive.git
$ cd prodive
$ bundle install
$ rails db:create && rails db:migrate
$ rails s
```
